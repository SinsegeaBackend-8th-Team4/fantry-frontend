# Fantry

## 프로젝트 소개

한정판 중고 굿즈 거래 서비스

## 프로젝트 기술

#### 프레임워크 - Vue3

- 라우터 - vue-router
- 상태관리 - Pinia

## 빌드도구

- [Vite](https://vite.dev/)
  - [Vite Configuration Reference](https://vite.dev/config/).

## 프로젝트 Setup

### 사전 준비

- 자바스크립트 런타임 설치 (Node.js 혹은 Bun)
  - [Node.js](https://nodejs.org/ko/) (권장)
  - [Bun](https://bun.com/)



### 모듈설치

Node.js 사용시

```sh
npm install
```

Bun 사용시

```sh
bun install
```

### 개발을 위한 컴파일 및 Hot-Reload 모드

Node.js 사용시

```sh
npm run dev
```

Bun 사용시

```sh
bun dev
```

### 프로덕션 배포를 위한 컴파일 및 소스코드 압축

Node.js 사용시

```sh
npm run build
```

Bun 사용시

```sh
bun run build
```
